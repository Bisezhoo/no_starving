## 4. 剩余偏差与建议

| 编号 | 剩余偏差 | 影响 | 建议优先级 |
|------|----------|------|------------|
| G1 | 未执行真实 OpenRouter / TheMealDB / TheCocktailDB 在线冒烟 | 自动化测试无法证明当前网络、密钥和真实模型输出完全可用 | P0 |
| G2 | 外部 API adapter 未实现设计中的 5xx/网络错误重试一次 | 外部 API 短暂抖动时降级能力不足 | P1 |
| G3 | 本地 JSON 写入没有文件 `fsync` 和目录 `fsync` | 进程或系统异常时仍可能丢最新写入 | P1 |
| G4 | 多轮“第一个/刚才那个”没有确定性引用解析器 | 正常模型可用，但模型失误时只能依赖校验失败或追问 | P1 |
| G5 | Tool 参数修正次数缺少显式全局状态机 | 异常 LLM 可能在预算内多次尝试非法参数 | P1 |
| G6 | 画像提取覆盖仍较窄 | 素食、纯素、地区菜系等偏好提取主要依赖 LLM 或少量规则 | P2 |
